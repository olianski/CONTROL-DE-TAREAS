# 🚀 Acceso Rápido al Sistema de Tareas

## 📋 Opciones de Acceso Simplificado

### 1️⃣ **Para el Jefe (Tú)**
- **Doble clic** en `iniciar_sistema.bat` para iniciar la aplicación
- Ve a `http://localhost:5000` o usa el acceso directo

### 2️⃣ **Para el Empleado**
- **URL Directa:** `http://192.168.1.39:5000`
- **Acceso Directo:** Doble clic en `acceso_empleado.url`
- **QR Code:** Escanea el archivo `acceso_sistema_qr.png` con el celular

### 3️⃣ **Credenciales Guardadas**
La nueva página de inicio ya incluye las credenciales precargadas:
- **Jefe:** usuario: `jefe`, contraseña: `admin123`
- **Empleado:** usuario: `empleado1`, contraseña: `emp123`

## 🎯 **Flujo Simplificado**

### Para Iniciar:
1. **Tú (Jefe):** Doble clic en `iniciar_sistema.bat`
2. **Empleado:** Abre `http://192.168.1.39:5000` o escanea el QR

### Para Usar:
1. Elige tu rol (Jefe/Empleado)
2. Las credenciales se autocompletan
3. Haz clic en "Iniciar Sesión"

## 📱 **Acceso Móvil**
- Escanea el QR con tu teléfono
- Guarda el acceso directo en la pantalla de inicio

## 🔧 **Si hay problemas:**
- Asegúrate de que ambos estén en la misma red WiFi
- Verifica que el firewall permita el puerto 5000
- Reinicia el sistema con `iniciar_sistema.bat`

**¡Listo! Ahora el acceso es mucho más simple y rápido.** 🎉
